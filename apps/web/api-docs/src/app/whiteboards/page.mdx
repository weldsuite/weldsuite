# Whiteboards

Whiteboards are collaborative infinite-canvas boards linked to a WeldFlow project. Each board stores its drawing data as a structured blob. Mutations require the `whiteboards:write` scope; reads require `whiteboards:read`. {{ className: 'lead' }}

<ResourceVersionBanner />

---

## The whiteboard model

### Properties

<Properties>
  <Property name="id" type="string">Unique identifier (e.g. `wb_abc123`).</Property>
  <Property name="name" type="string">Whiteboard display name.</Property>
  <Property name="projectId" type="string">ID of the parent project.</Property>
  <Property name="data" type="object">Canvas drawing data blob (format is internal and subject to change).</Property>
  <Property name="createdAt" type="timestamp">When the whiteboard was created.</Property>
  <Property name="updatedAt" type="timestamp">When the whiteboard was last updated.</Property>
</Properties>

## List whiteboards {{ tag: 'GET', label: '/v1/whiteboards' }}

<Row>
  <Col>

    Retrieve a cursor-paginated list of whiteboards.

    ### Optional query parameters

    <Properties>
      <Property name="limit" type="integer">Items per page (max 100).</Property>
      <Property name="cursor" type="string">Pagination cursor.</Property>
      <Property name="projectId" type="string">Filter by project.</Property>
    </Properties>

  </Col>
  <Col sticky>

    <CodeGroup title="Request" tag="GET" label="/v1/whiteboards">

    ```bash {{ title: 'cURL' }}
    curl -G https://api.weldsuite.org/v1/whiteboards \
      -H "Authorization: Bearer wsk_your_api_key" \
      -d projectId=proj_abc123
    ```

    </CodeGroup>

    ```json {{ title: 'Response' }}
    {
      "data": [
        {
          "id": "wb_abc123",
          "name": "Main Whiteboard",
          "projectId": "proj_abc123"
        }
      ],
      "pagination": { "totalCount": 2, "hasMore": false, "cursor": null }
    }
    ```

  </Col>
</Row>

## Create a whiteboard {{ tag: 'POST', label: '/v1/whiteboards' }}

<Row>
  <Col>

    Create a new whiteboard.

    ### Optional attributes

    <Properties>
      <Property name="name" type="string">Display name (defaults to `Main Whiteboard`).</Property>
      <Property name="projectId" type="string">Parent project ID.</Property>
      <Property name="data" type="object">Initial canvas data.</Property>
    </Properties>

  </Col>
  <Col sticky>

    <CodeGroup title="Request" tag="POST" label="/v1/whiteboards">

    ```bash {{ title: 'cURL' }}
    curl https://api.weldsuite.org/v1/whiteboards \
      -H "Authorization: Bearer wsk_your_api_key" \
      -H "Content-Type: application/json" \
      -d '{
        "name": "Architecture Diagram",
        "projectId": "proj_abc123"
      }'
    ```

    </CodeGroup>

    ```json {{ title: 'Response' }}
    {
      "data": {
        "id": "wb_new456",
        "name": "Architecture Diagram",
        "projectId": "proj_abc123",
        "createdAt": "2024-12-01T14:00:00Z",
        "updatedAt": "2024-12-01T14:00:00Z"
      }
    }
    ```

  </Col>
</Row>
