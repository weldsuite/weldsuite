# Drafts

Drafts are unsent message drafts scoped to a WeldChat channel. A user can have one active draft per channel. Mutations require the `chat_drafts:write` scope; reads require `chat_drafts:read`. {{ className: 'lead' }}

<ResourceVersionBanner />

---

## The draft model

### Properties

<Properties>
  <Property name="id" type="string">Unique identifier (e.g. `cdraft_abc123`).</Property>
  <Property name="channelId" type="string">ID of the channel the draft belongs to.</Property>
  <Property name="body" type="string">Draft body text.</Property>
  <Property name="createdAt" type="timestamp">When the draft was created.</Property>
  <Property name="updatedAt" type="timestamp">When the draft was last updated.</Property>
</Properties>

---

## List drafts {{ tag: 'GET', label: '/v1/chat-drafts' }}

<Row>
  <Col>

    Retrieve a cursor-paginated list of the current user's chat drafts.

    ### Optional query parameters

    <Properties>
      <Property name="limit" type="integer">Items per page (max 100).</Property>
      <Property name="cursor" type="string">Pagination cursor.</Property>
      <Property name="channelId" type="string">Filter by channel.</Property>
    </Properties>

  </Col>
  <Col sticky>

    <CodeGroup title="Request" tag="GET" label="/v1/chat-drafts">

    ```bash {{ title: 'cURL' }}
    curl -G https://api.weldsuite.org/v1/chat-drafts \
      -H "Authorization: Bearer wsk_your_api_key" \
      -d channelId=channel_abc123
    ```

    </CodeGroup>

    ```json {{ title: 'Response' }}
    {
      "data": [
        {
          "id": "cdraft_abc123",
          "channelId": "channel_abc123",
          "body": "Here's the updated plan..."
        }
      ],
      "pagination": { "totalCount": 1, "hasMore": false, "cursor": null }
    }
    ```

  </Col>
</Row>
