# Project Labels

Project labels are coloured tags used to categorise tasks within a WeldFlow project. Mutations require the `project_labels:write` scope; reads require `project_labels:read`. {{ className: 'lead' }}

<ResourceVersionBanner />

---

## The project label model

### Properties

<Properties>
  <Property name="id" type="string">Unique identifier (e.g. `plabel_abc123`).</Property>
  <Property name="name" type="string">Label display name (max 100 characters).</Property>
  <Property name="color" type="string">Brand colour (hex or named).</Property>
  <Property name="projectId" type="string">ID of the scoping project (optional, workspace-wide when absent).</Property>
  <Property name="createdAt" type="timestamp">When the label was created.</Property>
  <Property name="updatedAt" type="timestamp">When the label was last updated.</Property>
</Properties>

## List project labels {{ tag: 'GET', label: '/v1/project-labels' }}

<Row>
  <Col>

    Retrieve a cursor-paginated list of project labels.

    ### Optional query parameters

    <Properties>
      <Property name="limit" type="integer">Items per page (max 100).</Property>
      <Property name="cursor" type="string">Pagination cursor.</Property>
      <Property name="projectId" type="string">Filter by project.</Property>
    </Properties>

  </Col>
  <Col sticky>

    <CodeGroup title="Request" tag="GET" label="/v1/project-labels">

    ```bash {{ title: 'cURL' }}
    curl -G https://api.weldsuite.org/v1/project-labels \
      -H "Authorization: Bearer wsk_your_api_key" \
      -d projectId=proj_abc123
    ```

    </CodeGroup>

    ```json {{ title: 'Response' }}
    {
      "data": [
        {
          "id": "plabel_abc123",
          "name": "Bug",
          "color": "#EF4444",
          "projectId": "proj_abc123"
        }
      ],
      "pagination": { "totalCount": 8, "hasMore": false, "cursor": null }
    }
    ```

  </Col>
</Row>

## Create a project label {{ tag: 'POST', label: '/v1/project-labels' }}

<Row>
  <Col>

    Create a new project label.

    ### Required attributes

    <Properties>
      <Property name="name" type="string">Label display name (max 100 characters).</Property>
    </Properties>

    ### Optional attributes

    <Properties>
      <Property name="color" type="string">Brand colour.</Property>
      <Property name="projectId" type="string">Scoping project ID.</Property>
    </Properties>

  </Col>
  <Col sticky>

    <CodeGroup title="Request" tag="POST" label="/v1/project-labels">

    ```bash {{ title: 'cURL' }}
    curl https://api.weldsuite.org/v1/project-labels \
      -H "Authorization: Bearer wsk_your_api_key" \
      -H "Content-Type: application/json" \
      -d '{
        "name": "Feature Request",
        "color": "#8B5CF6",
        "projectId": "proj_abc123"
      }'
    ```

    </CodeGroup>

    ```json {{ title: 'Response' }}
    {
      "data": {
        "id": "plabel_new456",
        "name": "Feature Request",
        "color": "#8B5CF6",
        "projectId": "proj_abc123",
        "createdAt": "2024-12-01T14:00:00Z",
        "updatedAt": "2024-12-01T14:00:00Z"
      }
    }
    ```

  </Col>
</Row>
