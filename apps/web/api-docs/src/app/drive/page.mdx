# Drive

The Drive resource provides a unified, read-only view across all WeldDrive content, files from Drive, projects, documents, whiteboards, mail attachments, recordings, and more. It also exposes per-source storage counts. For creating, updating, or deleting files see [Files](/files); for folders see [Folders](/folders). Reads require the `drive:read` scope. {{ className: 'lead' }}

## List all items {{ tag: 'GET', label: '/v1/drive/all' }}

<Row>
  <Col>

    Retrieve a cursor-paginated unified feed of files and folders across all sources. Results are ordered newest-first.

    ### Optional query parameters

    <Properties>
      <Property name="search" type="string">Full-text search across file names.</Property>
      <Property name="type" type="string">Filter by semantic file type (e.g. `image`, `document`).</Property>
      <Property name="source" type="string">Filter by source (e.g. `drive`, `projects`, `whiteboards`, `mail`, `meetings`).</Property>
      <Property name="sortBy" type="string">Sort field.</Property>
      <Property name="sortOrder" type="string">`asc` or `desc`.</Property>
    </Properties>

  </Col>
  <Col sticky>

    <CodeGroup title="Request" tag="GET" label="/v1/drive/all">

    ```bash {{ title: 'cURL' }}
    curl -G https://api.weldsuite.org/v1/drive/all \
      -H "Authorization: Bearer wsk_your_api_key" \
      -d source=drive \
      -d sortOrder=desc
    ```

    </CodeGroup>


## Storage stats {{ tag: 'GET', label: '/v1/drive/stats' }}

<Row>
  <Col>

    Retrieve per-source file counts and totals for the workspace.

  </Col>
  <Col sticky>

    <CodeGroup title="Request" tag="GET" label="/v1/drive/stats">

    ```bash {{ title: 'cURL' }}
    curl https://api.weldsuite.org/v1/drive/stats \
      -H "Authorization: Bearer wsk_your_api_key"
    ```

    </CodeGroup>

    ```json {{ title: 'Response' }}
    {
      "data": {

        "recentCount": 12,

          "documents": 60,
          "whiteboards": 18,

          "meetings": 25
    ```
