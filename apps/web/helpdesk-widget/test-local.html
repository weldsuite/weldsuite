Welcome to Acme Corp

This is a simulated website to test the helpdesk widget in a realistic environment. The widget loads from your local dev server.

How it works

This page creates iframes pointing directly to your local Vite dev server (localhost:3100). Any changes you make to the widget code are picked up instantly via HMR — no SDK rebuild needed. The page mimics the same postMessage protocol the real SDK uses.

Quick start

1. Run pnpm dev in apps/web/helpdesk-widget to start the widget dev server on port 3100.
2. Open this file in a browser (or serve it with any static server).
3. The launcher button should appear in the bottom-right corner.
4. Use the dev toolbar to control the widget, change the widget ID, or view the message console.
