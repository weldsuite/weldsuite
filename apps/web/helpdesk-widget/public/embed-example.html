🎯 Helpdesk Widget - Embedding Guide

Integrate the helpdesk widget into your website with custom configuration

📦 Basic Embed (Default Configuration)

The simplest way to embed the widget with default settings:

<iframe
  src="http://localhost:3100/widget"
  style="position: fixed; bottom: 0; right: 0; width: 100%; height: 100%; border: none; z-index: 999999;"
  allow="clipboard-write"
></iframe>

🎨 Custom Theme Configuration

Customize colors, starting page, and enabled features:

<iframe
  src="http://localhost:3100/widget?launcherColor=%23667eea&startingPage=messages&enabledPages=home,messages,help,status"
  style="position: fixed; bottom: 0; right: 0; width: 100%; height: 100%; border: none; z-index: 999999;"
  allow="clipboard-write"
></iframe>

🚀 Advanced Configuration with All Options

<iframe
  src="http://localhost:3100/widget?defaultOpen=true&launcherColor=%23000000&startingPage=home&enabledPages=home,messages,help,status,changelog,news,feedback,appointments&borderRadius=24px&disableBackNavigation=false"
  style="position: fixed; bottom: 0; right: 0; width: 100%; height: 100%; border: none; z-index: 999999;"
  allow="clipboard-write"
></iframe>

⚙️ Configuration Parameters

Parameter Type Default Description
defaultOpen boolean false Opens the widget automatically on page load
launcherColor color (hex) #000000 Background color of the launcher button (URL encode: %23 for #)
startingPage string home Page to show when widget opens: home, messages, help, status, changelog, news, feedback, appointments, announcements, events, parcel-tracking
enabledPages string (comma-separated) all pages Pages to enable in the widget navigation
borderRadius CSS value 20px Border radius for the launcher button
disableBackNavigation boolean false Disables back navigation between views
primaryColor color (hex) - Primary theme color (URL encode: %23 for #)
backgroundColor color (hex) - Background color (URL encode: %23 for #)
textColor color (hex) - Text color (URL encode: %23 for #)

📝 Available Pages

  • home - Main dashboard with all features
  • messages - View conversation history
  • help - FAQ and knowledge base
  • status - System status dashboard
  • changelog - Product updates and releases
  • news - Company news and announcements
  • feedback - Submit feedback and ideas
  • appointments - Book appointments with support
  • announcements - System announcements
  • events - Upcoming events
  • parcel-tracking - Track parcels and deliveries

💡 Pro Tips

  • Always URL encode color values (replace # with %23)
  • Use comma-separated values for enabledPages without spaces
  • Set z-index: 999999 to ensure widget appears above other content
  • Include allow="clipboard-write" for copy-paste functionality
  • The widget is fully responsive and works on mobile devices

🌐 JavaScript Integration

For dynamic configuration, you can use JavaScript to build the URL:

const config = {
  launcherColor: '#667eea',
  startingPage: 'messages',
  enabledPages: ['home', 'messages', 'help', 'status'],
  defaultOpen: false
};

const params = new URLSearchParams();
params.set('launcherColor', encodeURIComponent(config.launcherColor));
params.set('startingPage', config.startingPage);
params.set('enabledPages', config.enabledPages.join(','));
params.set('defaultOpen', config.defaultOpen.toString());

const iframe = document.createElement('iframe');
iframe.src = `http://localhost:3100/widget?${params.toString()}`;
iframe.style.cssText = 'position: fixed; bottom: 0; right: 0; width: 100%; height: 100%; border: none; z-index: 999999;';
iframe.setAttribute('allow', 'clipboard-write');
document.body.appendChild(iframe);

Note: Replace localhost:3100 with your production domain when deploying
