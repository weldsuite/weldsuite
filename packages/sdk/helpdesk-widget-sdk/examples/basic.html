🎯 Helpdesk Widget - Basic Example

This example demonstrates the most basic usage of the Helpdesk Widget SDK. The widget is initialized with just a widgetId parameter - that's it!

How it works:

The widget is loaded in an iframe from https://widget.welddesk.com. All configuration (colors, position, pages, etc.) is managed in your backoffice.

Widget ID: demo-widget-123
Configuration: Managed in backoffice
